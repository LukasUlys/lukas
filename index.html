Gitas yra populiariausa kodo vesijavimo programa.

Git komandos

  • init - sukuria gito repozitorija
  • add - prideda nurodytus failus prie commitui paruostu failu
  • status - nurodo informacija apie tavo lokalio gito repozitorijos statusa
  • commit -m "zinute" - sukuria kodo versija is stagintu failu ir prideda zinute apie pakitimus (zinute bandyti islaikyti ne ilgesne negu 50 simboliu)
  • push - is lokalios repozitorijos commitu nusiuntimas i isorine repozitorija
  • pull - is isorines repozitorijos comiitu atsisiuntimas i lokalia repozitorija.
  • remote add origin https:// ... - isorines repozitorijos prijungimas prie lokalios repozitorijos
  • log - commitu sarasas
